# Capturas recomendadas

Este proyecto es principalmente CLI, así que no necesita muchas capturas. Aun así, para GitHub queda bien añadir imágenes limpias.

## Capturas útiles

```text
assets/screenshots/01-terminal-run.png
assets/screenshots/02-csv-preview.png
assets/screenshots/03-project-structure.png
```

## Qué mostrar

### `01-terminal-run.png`

Terminal ejecutando:

```powershell
python main.py
```

Ideal si se ven logs de offsets, scrolls y productos acumulados.

### `02-csv-preview.png`

Vista del CSV final abierto en Excel, LibreOffice o VS Code.

### `03-project-structure.png`

Estructura del proyecto en VS Code.

## Recomendación

No mostrar rutas personales, datos sensibles ni capturas enormes. Mejor pocas capturas claras que muchas imágenes repetidas.
