# Limitaciones

Este scraper trabaja sobre una web pública y dinámica. Eso implica que puede romperse si cambia el marcado, los selectores o la estrategia de carga.

## Limitaciones actuales

- No usa API oficial.
- Depende de Chrome y Selenium.
- Puede verse afectado por cambios de cookies, banners o protección anti-bot.
- No está diseñado para scraping masivo.
- El precio antiguo se calcula a partir del precio actual y descuento cuando ambos datos son coherentes.
- No incluye todavía tests automatizados del parser.
- No tiene CLI con argumentos.

## Uso responsable

El proyecto debe ejecutarse con límites razonables y sin sobrecargar la web objetivo. El objetivo es educativo y de portfolio técnico.
